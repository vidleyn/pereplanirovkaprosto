# Планировщик - Инструкция по интеграции

Эта директория предназначена для размещения планировщика.

## Структура

После интеграции здесь должна быть следующая структура:

```
features/planner/
├── src/
│   ├── legacy/          # Старый код (если есть)
│   ├── scripts/         # Скрипты планировщика
│   ├── styles/          # Стили планировщика
│   ├── ui/              # UI компоненты
│   │   └── components/
│   ├── utils/           # Утилиты
│   ├── App.tsx          # Главный компонент планировщика
│   └── styles.css       # Главный файл стилей
├── models/              # 3D модели (если они не в public/)
├── public/              # Публичные ресурсы планировщика
└── index.ts             # Экспорт главного компонента

```

## Шаги интеграции

1. Скопируйте исходный код планировщика в эту директорию
2. Объедините зависимости из `package.json` планировщика в основной `frontend/package.json`
3. Убедитесь, что все пути к ресурсам корректны
4. Импортируйте компонент в `src/pages/PlannerPage.tsx`

## Заметки

- Модели могут быть размещены в `frontend/public/planner/models/` (уже есть такая структура)
- Публичные ресурсы планировщика должны быть доступны через `/planner/`
